# Operadores Aritméticos

- pós-fixado: exp++ ou exp--
- prefixado: ++exp ou --exp
- aritmético: +, -, *, / e %
- atribuição: =, +=, -=, *=, /= e %=

## Exemplos

- prefixado: 
***int i = ++k;***
*i = k + 1*

- pós-fixado:
***int j = k--***
*j = k;*
*k = k - 1;*

- atribuição:
***k %= 2***
*k = k % 2*
